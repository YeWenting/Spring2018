Quality: About This Data Set

Wenting's Location and Cellphone Usage Dataset

Is the data complete?

The data match the questions I want to answer, but there are some missing data. Since I aggregated the data by minute, I found out the majority time don't have any data.

Is the data coherent?

There are some points not in the main road, the reason might be I took the bus to the supermarket and the signal is not stable.

The overall distribution is consistent with my life style.

Is the data correct?

1. I found out the timeframe for screen and location tables can not be perfectly matched. The reason is because aware sample the data based on different frequency by default. To align them, I loose the condition to minute level, using the condition "int([timestamp]/60000) = int([S.timestamp]/60000)". Here, the 60000 is 60 (minute to seconds) * 1000 (second to milliseconds).

2. There are some invalid numbers for screen status, I removed all the numbers that are not among 0 to 3.
